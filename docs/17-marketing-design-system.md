# Marketing Design System: School ERP SaaS

This document outlines the visual design system for the School ERP marketing website to ensure a premium, trust-heavy, and high-converting experience.

## 1. Typography
We use **Inter** (via Google Fonts or system default) for a clean, modern SaaS look.

| Element | Scale | Weight | Use Case |
|---|---|---|---|
| Heading 1 | 4rem (64px) | 800 (ExtraBold) | Hero Banner |
| Heading 2 | 2.5rem (40px) | 700 (Bold) | Section Headers |
| Heading 3 | 1.5rem (24px) | 600 (SemiBold) | Feature Titles |
| Subheading | 1.25rem (20px) | 400 (Regular) | Hero Subtext |
| Body | 1rem (16px) | 400 (Regular) | Paragraphs |
| Caption | 0.875rem (14px) | 500 (Medium) | Small Text, Tags |

## 2. Color Palette
A premium Indigo/Slate theme that conveys technology and trust.

| Name | HSL Value | Use Case |
|---|---|---|
| Primary | `221.2 83.2% 53.3%` | Primary Buttons, Accents |
| Background | `222.2 84% 4.9%` (Dark Mode Friendly) | Page background |
| Foreground | `210 40% 98%` | Primary text |
| Muted | `217.2 32.6% 17.5%` | Subtle cards, borders |
| Accent | `210 40% 96.1%` | Hover states |

## 3. Layout Grid
- **Max Width**: `80rem` (1280px) for standard containers.
- **Max Width Small**: `48rem` (768px) for blog posts and single columns.
- **Section Spacing**: `6rem` to `10rem` (96px to 160px) for vertical breathing room.
- **Gutter**: `1.5rem` (24px).

## 4. Components Style
- **Border Radius**: `0.75rem` (12px) for cards, `0.5rem` (8px) for buttons.
- **Shadows**: Soft, multi-layered shadows for depth.
- **Borders**: Thin, high-contrast borders (`border-slate-800`).
- **Gradients**: Subtle radial glows behind high-impact sections.

## 5. Tone & Voice
- **Professional**: "School Operating System", not "School App".
- **Trustworthy**: Emphasize audit logs, compliance, and safety.
- **Clear**: Focus on ROI and efficiency (fee collection, time saved).
